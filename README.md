# Poe Tauri

Poe Tauri is a lightweight Path of Exile companion overlay built with Tauri + React.
It helps you follow a leveling guide and track your progress while you play.
The goal of this project is to provide a Path of Exile overlay tool that works well on Wayland.

## Acknowledgments

This project is inspired by, and builds on ideas from, the following community tools:

- [Exile-UI](https://github.com/Lailloken/Exile-UI): the main inspiration for the overlay UX, and for reusing its
  leveling guide parsing logic.
- [Path of Building](https://github.com/PathOfBuildingCommunity/PathOfBuilding): build-planning data.
- [exile-leveling](https://github.com/HeartofPhos/exile-leveling): the original leveling guide.

## How to use

Run the app, load the default leveling guide, import your pob and click on show overlay. Keep it open while Path of
Exile is running.
